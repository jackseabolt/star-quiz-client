Why StarQuiz?
-------------

For local use
--------

```bash
# Clone repository
git clone https://github.com/jackseabolt/star-quizes-client.git

# Change directory
cd star-quizes-client

# Install NPM dependencies
npm install

# Start the server
npm start
```
